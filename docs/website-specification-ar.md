## 1) نظرة عامة على المشروع

يهدف هذا المشروع إلى إنشاء موقع إلكتروني احترافي متكامل لشركة متخصصة في تركيب أنظمة الطاقة الشمسية. الموقع سيكون الواجهة الرقمية الرئيسية للشركة ويعمل كأداة تسويقية قوية لجذب العملاء المحتملين وعرض الخدمات والأنظمة المتاحة.

سيتميز الموقع بتصميم عصري مع خلفية ثلاثية الأبعاد تعرض خلايا شمسية متحركة، مما يعطي انطباعًا احترافيًا وتقنيًا عاليًا. كما سيحتوي على لوحة تحكم كاملة لإدارة كل محتوى الموقع.

- **اسم المشروع:** موقع شركة الطاقة الشمسية
- **نوع المشروع:** موقع تسويقي + لوحة تحكم إدارية
- **اللغات:** عربي (أساسي) + إنجليزي
- **المنصات المرتبطة:** TikTok, Facebook, Instagram

## 2) أهداف الموقع

### 2.1 الأهداف التسويقية
- عرض خدمات الشركة بشكل احترافي وجذاب.
- بناء ثقة العملاء من خلال عرض المشاريع السابقة والشهادات.
- تسهيل التواصل مع العملاء المحتملين عبر نماذج الاتصال.
- تحسين ظهور الشركة في محركات البحث (SEO).
- ربط الموقع بحسابات التواصل الاجتماعي لزيادة التفاعل.

### 2.2 الأهداف التقنية
- كود نظيف ومنظم 100% بمعايير Clean Code.
- أمان عالٍ جدًا ضد جميع أنواع الهجمات.
- سرعة تحميل ممتازة (أقل من 3 ثوانٍ).
- تصميم متجاوب يعمل على جميع الأجهزة.
- لوحة تحكم شاملة لإدارة كامل الموقع.

## 5) تفاصيل كل صفحة ومحتواها

### 5.1 الصفحة الرئيسية (Landing Page)
- **Hero Section:** خلفية 3D لخلايا شمسية متحركة مع عنوان رئيسي جذاب وزر CTA (اطلب عرض سعر).
- **قسم الأنظمة الثلاثة:** عرض مختصر لكل نظام (Grid, Off-Grid, Hybrid) مع أيقونات ورابط لصفحة التفاصيل.
- **قسم الأرقام والإنجازات:** عدد المشاريع المنجزة، القدرة الكلية المركبة، عدد العملاء، سنوات الخبرة (عداد متحرك).
- **قسم آراء العملاء:** شهادات من عملاء سابقين مع صورهم وأسمائهم (Slider/Carousel).
- **قسم آخر المشاريع:** عرض 3-4 مشاريع حديثة من معرض الأعمال.
- **قسم الشركاء والعلامات التجارية:** شعارات الماركات المستخدمة (Jinko, Canadian Solar, Huawei, إلخ).
- **قسم CTA نهائي:** دعوة لاتخاذ إجراء مع رقم الهاتف ونموذج سريع.
- **Footer:** روابط سريعة، معلومات التواصل، روابط السوشيال ميديا.

### 5.2 صفحة من نحن (About)
- قصة الشركة ومسيرتها.
- الرؤية والرسالة والقيم.
- فريق العمل مع صورهم ومناصبهم.
- الشهادات والاعتمادات.
- Timeline لمسيرة الشركة.

### 5.3 صفحة الخدمات (Services)
صفحة محورية تعرض الأنظمة الثلاثة بالتفصيل مع مقارنة بينها. كل نظام يحتوي على وصف كامل، مميزات، عيوب، حالات الاستخدام المناسبة، ومخطط توضيحي.

### 5.4 معرض الأعمال (Portfolio)
- شبكة صور (Grid Gallery) للمشاريع المنجزة.
- فلترة حسب نوع النظام (Grid/Off-Grid/Hybrid).
- فلترة حسب الحجم (سكني/تجاري/صناعي).
- صفحة تفصيلية لكل مشروع تتضمن: الصور، الموقع، القدرة، نوع النظام، تاريخ التنفيذ، شهادة العميل.
- عرض الصور بشكل Lightbox عند الضغط عليها.

### 5.5 المدونة (Blog)
- مقالات تعليمية عن الطاقة الشمسية (مهمة جدًا للـ SEO).
- تصنيفات: تعليمي، أخبار، نصائح، تقنية.
- بحث في المقالات.
- مقالات مرتبطة في نهاية كل مقال.
- مشاركة على السوشيال ميديا.
- وقت القراءة المتوقع.

### 5.6 حاسبة التوفير (Calculator)
**المدخلات:**
- متوسط فاتورة الكهرباء الشهرية.
- مساحة السطح المتاحة (أو عدد الألواح).
- نوع النظام المطلوب.
- الموقع الجغرافي (لحساب ساعات الشمس).

**المخرجات:**
- التكلفة التقديرية.
- التوفير الشهري.
- فترة استرداد الاستثمار.
- كمية CO2 الموفرة.

زر: **(اطلب عرض سعر دقيق)** يحول للتواصل مع البيانات المدخلة.

### 5.7 تواصل معنا (Contact)
- نموذج اتصال: الاسم، الهاتف، البريد، نوع الخدمة، الرسالة.
- خريطة Google Maps لموقع الشركة.
- أرقام الهاتف والواتساب.
- البريد الإلكتروني.
- ساعات العمل.
- روابط السوشيال ميديا.

## 6) أنظمة الطاقة الشمسية الثلاثة

### 6.1 نظام On-Grid (متصل بالشبكة)
**التعريف:** نظام طاقة شمسية متصل مباشرة بشبكة الكهرباء العامة.

**المكونات:**
- ألواح شمسية (Solar Panels)
- إنفرتر متصل بالشبكة (Grid-Tie Inverter)
- عداد ثنائي الاتجاه (Bi-directional Meter)
- نظام مراقبة (Monitoring System)
- هيكل تثبيت (Mounting Structure)

**المميزات:**
- أقل تكلفة بين الأنظمة الثلاثة (لا يحتاج بطاريات)
- إمكانية بيع الفائض للشبكة (Net Metering)
- صيانة بسيطة وعمر افتراضي طويل (25+ سنة)
- فترة استرداد قصيرة (3-5 سنوات)

**العيوب:**
- لا يعمل عند انقطاع الكهرباء (لأسباب أمان)
- يعتمد كليًا على وجود شبكة كهرباء

**الأنسب لـ:**
- المنازل والمحلات في مناطق بها شبكة كهرباء مستقرة
- من يريد تقليل فاتورة الكهرباء بأقل تكلفة ممكنة

### 6.2 نظام Off-Grid (منفصل عن الشبكة)
**التعريف:** نظام مستقل تمامًا عن شبكة الكهرباء.

**المكونات:**
- ألواح شمسية (Solar Panels)
- منظم شحن (Charge Controller - MPPT/PWM)
- بنك بطاريات (Battery Bank - Lithium/Lead-Acid)
- إنفرتر مستقل (Off-Grid Inverter)
- هيكل تثبيت (Mounting Structure)
- نظام مراقبة (Monitoring System)

**المميزات:**
- استقلالية تامة عن شبكة الكهرباء
- مثالي للمناطق النائية التي لا تصلها الشبكة
- لا يتأثر بانقطاعات الكهرباء
- حل ممتاز للمزارع والمناطق الريفية

**العيوب:**
- تكلفة أعلى بسبب البطاريات
- البطاريات تحتاج استبدال كل 5-10 سنوات
- يحتاج تصميم دقيق لتغطية الاستهلاك الكامل
- لا يمكن بيع الفائض

**الأنسب لـ:**
- المناطق النائية بدون شبكة كهرباء
- المزارع والمشاريع الزراعية
- من يريد استقلالية كاملة عن الشبكة

### 6.3 نظام Hybrid (هجين)
**التعريف:** يجمع بين مزايا النظامين السابقين.

**المكونات:**
- ألواح شمسية (Solar Panels)
- إنفرتر هجين (Hybrid Inverter)
- بنك بطاريات (Battery Bank)
- عداد ثنائي الاتجاه (Bi-directional Meter)
- نظام مراقبة ذكي (Smart Monitoring System)
- هيكل تثبيت (Mounting Structure)

**المميزات:**
- يعمل حتى عند انقطاع الكهرباء (بفضل البطاريات)
- إمكانية بيع الفائض للشبكة
- أقصى استفادة من الطاقة الشمسية
- مرونة عالية في إدارة الطاقة
- يمكن برمجته لاستخدام الشبكة في ساعات الذروة فقط

**العيوب:**
- الأعلى تكلفة بين الأنظمة الثلاثة
- يحتاج إنفرتر خاص أغلى سعرًا
- تعقيد أكبر في التركيب والإعدادات

**الأنسب لـ:**
- من يريد أقصى حماية ومرونة
- المناطق التي تعاني من انقطاعات متكررة
- من يريد الاستفادة القصوى من استثماره

### 6.4 جدول مقارنة الأنظمة الثلاثة

| المعيار | On-Grid | Off-Grid | Hybrid |
|---|---|---|---|
| التكلفة | منخفضة ●○○ | متوسطة-عالية ●●○ | عالية ●●● |
| البطاريات | لا يحتاج | ضرورية | اختيارية/ضرورية |
| يعمل بدون شبكة | لا | نعم | نعم |
| بيع الفائض | نعم | لا | نعم |
| الصيانة | بسيطة | متوسطة | متوسطة |
| فترة الاسترداد | 3-5 سنوات | 5-8 سنوات | 4-7 سنوات |
| التعقيد | بسيط | متوسط | معقد |
| الأفضل لـ | مناطق مستقرة | مناطق نائية | أقصى مرونة |

## 7) التصميم وتجربة المستخدم (UI/UX)

### 7.1 الهوية البصرية
تصميم عصري داكن (Dark Theme) مع لمسات ذهبية/برتقالية.

| العنصر | القيمة | الملاحظة |
|---|---|---|
| اللون الأساسي | `#F59E0B` | يرمز للشمس والطاقة |
| اللون الثانوي | `#0F172A` | خلفية أنيقة |
| لون النجاح | `#22C55E` | للأرقام الإيجابية |
| لون التحذير | `#EF4444` | للتنبيهات |
| لون المعلومات | `#3B82F6` | للروابط والأزرار الثانوية |
| الخط العربي | Tajawal | واضح وعصري |
| الخط الإنجليزي | Outfit | هندسي ونظيف |
| حجم النص الأساسي | 16px / 1rem | قراءة مريحة |
| الزوايا | 8px - 16px | زوايا ناعمة |
| الظلال | Soft shadows + Glow | عمق بصري |

### 7.2 مبادئ UX
- قائمة واضحة ثابتة (Sticky Navbar) بحد أقصى 7 عناصر.
- أي معلومة على بعد 3 نقرات كحد أقصى.
- CTA بارزة في كل صفحة.
- سرعة تحميل أقل من 3 ثوانٍ + Lazy Loading.
- Mobile-First.
- Accessibility كاملة (تباين/Alt Text/Keyboard).
- Smooth Scrolling + Scroll Animations.
- زر واتساب عائم في كل الصفحات.

### 7.3 تصميم المكونات
- **Buttons:** Primary / Secondary / Ghost.
- **Cards:** Glass Morphism.
- **Forms:** Floating Label + تحقق لحظي.
- **Tables:** ألوان متناوبة + فرز وبحث.
- **Toasts:** نجاح/خطأ/تحذير بإغلاق تلقائي.

## 8) خلفية ثلاثية الأبعاد (3D Background)

### 8.1 المواصفات التقنية
| العنصر | التفاصيل |
|---|---|
| المكتبة | Three.js أو React Three Fiber |
| المشهد | صفوف ألواح شمسية على أرضية داكنة |
| الإضاءة | Directional + Ambient + Rim Light |
| الحركة | دوران بطيء للكاميرا + Parallax خفيف |
| الأداء | Pixel Ratio ≤ 2 + LOD |
| Fallback | صورة ثابتة للأجهزة غير الداعمة لـ WebGL |

### 8.2 عناصر المشهد
- 40-60 لوح شمسي ضمن 5-7 صفوف.
- كل لوح: إطار معدني + خلايا زرقاء داكنة + خطوط شبكة + عمود تثبيت + قاعدة.
- إضاءة ديناميكية (اختياري).
- Particles خفيفة تمثل أشعة الشمس.
- العنوان الرئيسي وزر CTA فوق المشهد مباشرة.

## 12) الأمان والحماية (Security)

### 12.1 المصادقة والتفويض
- NextAuth.js + بريد/كلمة مرور.
- bcrypt (12+ rounds).
- JWT قصير المدة + Refresh Token.
- Middleware لحماية `/admin/*`.
- RBAC.
- قفل الحساب بعد 5 محاولات فاشلة.
- Auto Logout عند عدم النشاط.

### 12.2 حماية البيانات
- HTTPS/TLS 1.3 إجباري.
- Input Validation عبر Zod (عميل/خادم).
- حماية SQL Injection عبر Prisma.
- حماية XSS + CSP.
- حماية CSRF + SameSite Cookies.
- Encryption at Rest.
- فحص ملفات الرفع.

### 12.3 حماية الشبكة
- Rate Limiting.
- WAF.
- DDoS protection عبر CDN.
- CORS مقيّد.
- HTTP Security Headers.

### 12.4 أفضل الممارسات
- Environment Variables للأسرار.
- Generic Error Messages في الإنتاج.
- تحديث دوري للحزم.
- Security Audit دوري.
- نسخ احتياطية يومية.
- Security Logging.

| تابلت | `641px - 1024px` | عمودان وSidebar قابل للطي |
| لابتوب | `1025px - 1440px` | تصميم كامل |
| شاشة كبيرة | `> 1440px` | max-width مع هوامش |

- خلفية 3D تصبح صورة ثابتة على الموبايل.
- لوحة التحكم متجاوبة بالكامل.
- الجداول تتحول إلى بطاقات على الشاشات الصغيرة.
- Fluid Typography عبر `clamp()`.
